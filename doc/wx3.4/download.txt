== Java ==

The required Java runtime can be downloaded at [http://java.com/ java.com]. Make sure that you have got at least version 1.6&mdash;anything higher is okay too ;)

== SVN ==

If you'd like to throw an eye on wiki2xhtml: The source code is being managed with Subversion (SVN) on this repository: https://wiki2xhtml.svn.sourceforge.net/svnroot/wiki2xhtml/trunk/wiki2xhtml-current/

The code and the rest can be downloaded directly with svn<ref>[http://de.wikipedia.org/wiki/Subversion%20(Software) Subversion] (SVN) is a version cotrol tool. See also the free [http://svnbook.red-bean.com/ Subversion book].</ref> or with programs like 
* [http://rapidsvn.tigris.org/ RapidSVN] or
* [http://tortoisesvn.net/ TortoiseSVN]
Keep in mind that this version is in development and perhaps doesn't run stable.

Compile the project by running $$javac src/UserInterface.java$$; you need to have Java Development Kit 1.6 or higher installed. 

If you use Linux, you can generate the jar files by executing $$build.sh$$. ant<ref>[http://en.wikipedia.org/wiki/Apache_Ant Ant] is &ldquo;make for Java&rdquo;</ref> needs to be installed.
